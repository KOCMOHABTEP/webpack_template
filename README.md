# Webpack template for creating APPS
### Includes:
* Webpack 5
* PostCSS
* Pug


```bash
# Clone repository 
git clone https://github.com/KOCMOHABTEP/webpack_template

# Go to folder
cd webpack_template

# Install deps
npm install

# Start developing
npm run dev
```
